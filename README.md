# Hacker Terminal

A simple Python script that endlessly prints random characters in random colors to the terminal. Perfect for showcasing the hacker aesthetic in videos, short films, and creative projects.

Originally created this back in mid-2024 — about time I put it on GitHub.

## Usage

```bash
git clone https://github.com/poran-dip/hacker-terminal.git
cd hacker-terminal
python main.py
```

Stop anytime with `Ctrl+C`.
